Key Bindings

In this page we will list a couple of shortcuts we might use in Visual Studio Code

  1. cmd + ~        To open up terminal
  2. ctrl + G         Navigate to line
  3. cmd + k, cmd + [  Fold all subregions
  4. cmd + b        Toggle Sidebar
  5. cmd + K, Z      Zen mode

For more information click here to the key binding document for MACOSX
